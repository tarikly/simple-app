Red Hat GitHub Actions Demo

Leverage OpenShift Developer Tools and GitHub Actions

Red Hat GitHub Actions Workflow is designed to help you work seamlessly with Red Hat Developer tools and Openshift clusters.

This application used the RedHat OpenShift Starter Workflow to get an app running in minutes.

Together, we're super awesome (so is openshift commons)

Run an App on OpenShift in 4 Easy Steps

  1. Get a sandbox cluster https://developers.redhat.com/developer-sandbox
  2. Add the OpenShift starter workflow to your favourite container based app or fork this sample app and modify the OpenShift Starter Workflow for your environment.
  3. Configure the secrets for your OpenShift cluster and preferred container registry.
  4. Push changes to your app and the built-in workflow will have this application running in minutes.
